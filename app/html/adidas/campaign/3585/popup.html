ADIDAS CRAZYCOURT 2017

  • 등록선수여부란 대한농구협회(KBA), 한국농구연맹(KBL)에 등록되어 있는지를 묻는 것이며, 초 / 중 / 고 / 대학일반부 / 우먼스부의 경우 전원 X 표시 되어야 출전 가능
  • 경기 당일 현장에서 참가자의 신원을 확인할 수 있는 신분증(미성년자의 경우 여권, 사진이 첨부된 학생증, 의료보험증, 주민등록초본 중 택1)을 반드시 지참하여 제시해야 함(참가부분 대상 나이와 참가자 나이가 상이하거나, 신분증 미지참 시 대회참가가 불가함을 알려드립니다.)
  • 참가자 대표 회원정보의 주소지를 정확하게 업데이트 해주셔야 추후 경품 배송이 가능합니다. (주소지 미기입/오류로 인한 불이익에 대한 책임은 참가자 본인에게 있음을 명시합니다.)
  • 참가자 본인의 얼굴 사진(ex.증명사진)이미지 파일을 반드시 대회 사무국 이메일 (2017crazycourt@workerb.co.kr)로 반드시 송부해야 해야 합니다. 해당 사진은 ‘크레이지코트’ 모바일 사이트에서 추후 선수들의 다양한 경기 기록 제공 서비스를 위해 사용될 예정입니다.
